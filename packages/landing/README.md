# Firma-Sign Landing Page

A modern, compelling one-pager landing page for Firma-Sign - the decentralized document signing solution built on true Web3 principles.

## Overview

This landing page communicates Firma-Sign's unique value proposition: peer-to-peer document signing with blockchain certification. It's designed to appeal to both Web3 enthusiasts and non-technical users, emphasizing practical decentralization over complex technology.

## Design Philosophy

The landing page follows FirmaChain's established brand identity:

- Dark, modern aesthetic with blue accent colors
- Professional yet approachable tone
- Focus on the revolutionary P2P functionality
- Clear messaging about blockchain's role (certification only)
- Emphasis on user ownership and freedom

## Key Messages

1. **True P2P Technology**: Documents transfer directly between peers - no servers
2. **Blockchain for Certification**: Using blockchain for what it does best - permanent proof
3. **Forever Free Core**: Basic P2P signing will always be free
4. **Real Web3 Implementation**: Not just buzzwords, actual working decentralization

## Structure

The landing page includes:

- **Hero Section**: Powerful first impression with clear value proposition
- **Problem/Solution**: Why centralized signing is outdated
- **How It Works**: Simple 3-step process visualization
- **Features**: Six key benefits with icons
- **Comparison**: Traditional vs Firma-Sign
- **Technology**: Simple explanation of P2P and blockchain
- **Use Cases**: Who benefits and how
- **FAQs**: Addressing common concerns
- **CTA**: Strong closing with action buttons

## Development

### Prerequisites

- Node.js 18+
- pnpm

### Installation

```bash
# Install dependencies
pnpm install
```

### Development Server

```bash
# Start development server on port 3001
pnpm dev
```

### Build

```bash
# Build for production
pnpm build

# Preview production build
pnpm preview
```

### Type Checking

```bash
pnpm type-check
```

### Formatting

```bash
# Format code
pnpm format

# Check formatting
pnpm format:check
```

## Technology Stack

- **React 19**: Latest React with improved performance
- **Vite**: Fast build tool and dev server
- **Tailwind CSS v4**: Utility-first CSS framework
- **Framer Motion**: Smooth animations and interactions
- **TypeScript**: Type safety and better DX
- **Lucide React**: Beautiful, consistent icons

## Design System

The design system is aligned with FirmaChain's brand:

### Colors

- Primary: `#1050dd` (FirmaChain Blue)
- Secondary: `#316bff` (Lighter Blue)
- Background: `#121417` (Dark Charcoal)
- Text: White, grays for hierarchy

### Typography

- Headers: Metropolis (bold, impactful)
- Body: Lato (clean, readable)
- Monospace: Fira Code (for any code elements)

### Animations

- Subtle scroll animations
- Hover effects on interactive elements
- Gradient background shifts
- Smooth transitions throughout

## Content Strategy

1. **Web3 Enthusiasts**: Those who understand and value decentralization
2. **Business Users**: Looking for secure, cost-effective signing solutions
3. **Privacy-Conscious Users**: Concerned about data ownership
4. **Developers**: Interested in open-source solutions

- First truly P2P document signing solution
- Blockchain used only for certification (not storage)

- Open source and community-driven
